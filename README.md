# Nike Store Clone (Frontend Web Project)

A responsive **Nike Store Clone** built using **HTML, CSS, and JavaScript**, designed to simulate an e-commerce sneaker store experience. The project includes product listings, detailed product pages, a payment interface, and an admin-style dashboard layout for managing products, customers, and orders.
